Welcome to OptoBot's Documentation!
===================================
**OptoBot** is a package for implementing automated experimental optimisation 
using the `Opentrons OT-2 <https://opentrons.com/robots/ot-2>`_ liquid handling 
robot. 
Its aim is to enable lab scientists to implement automated experimental 
optimisation in their own work with minimal programming experience. 
Checkout the package below!

+ **Release Version**: 0.1.0
+ **GitHub**: `<https://github.com/nicolaegues/OptoBot>`_

- **Open Source**: MIT License
- **Python Versions**: 3.10, 3.11, 3.12
